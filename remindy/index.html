Remindy · Lock Screen Widgets

Simple, customizable information on your lock screen.

Remindy is a personal app focused on making lock screen widgets more useful and more personal. It starts simple: clean widgets, simple information, and lightweight customization. Over time, it will expand with more utility and personalization features.

About Remindy

Remindy is being built as a lightweight app to help users display useful information through lock screen widgets in a simple and visually pleasant way. The initial focus is clarity, customization, and quick access to information without opening the app constantly.

Lock screen widgets Simple setup Personalization More features coming later

Current Focus

The first version of Remindy is centered around providing a simple way to configure and display personalized widgets with useful information on the lock screen.

Simple widget customization

Configure widgets in a straightforward way without complicated setup flows.

Clean information display

Show useful information clearly and quickly, designed for at-a-glance viewing.

Lightweight experience

Keep the app fast, minimal, and focused on the most important use cases first.

Planned Features

Remindy starts small, but the vision is broader. Future versions may include more personalization, richer content, and utility-focused improvements.

More widget styles

Additional visual themes, layouts, and presentation options for different tastes.

Expanded information sources

New types of content and useful data displayed directly through widgets.

Better personalization

More control over what is shown, how it looks, and how it fits the user's needs.
